# Gemini AI Chatbot 🤖

A sleek and responsive AI chatbot inspired by Google Gemini's UI, built with HTML, CSS, and JavaScript. It delivers real-time intelligent responses powered by the Gemini API, with a clean interface designed for smooth and engaging conversations.

##  Features
-  Real-time AI responses using Google Gemini API
-  Typing indicator and loading animation
-  Copy response to clipboard
-  Light / Dark theme toggle
-  Chat history saved with LocalStorage

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Google Gemini API (gemini-2.5-flash)
